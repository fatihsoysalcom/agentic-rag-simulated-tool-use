# Agentic RAG Simulated Tool Use

This example demonstrates the core concept of Agentic RAG by simulating an AI agent that decides which retrieval 'tool' to use based on the user's query. Instead of a single, passive retrieval, the agent actively orchestrates different retrieval strategies (e.g., for definitions, comparisons, or general information) to provide a more targeted response.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
